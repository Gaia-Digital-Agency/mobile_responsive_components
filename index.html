Fluid Typography

This is lead text using fluid sizing with clamp(). It scales smoothly from mobile to desktop without breakpoint jumps.

All Heading Levels with Fluid Sizing

Heading 1 - clamp(2rem, 5vw + 1rem, 4rem)

Heading 2 - clamp(1.75rem, 4vw + 0.75rem, 3rem)

Heading 3 - clamp(1.5rem, 3vw + 0.5rem, 2.5rem)

Heading 4 - clamp(1.25rem, 2vw + 0.5rem, 2rem)

Heading 5 - clamp(1.125rem, 1.5vw + 0.5rem, 1.5rem)
Heading 6 - clamp(1rem, 1vw + 0.5rem, 1.25rem)

Regular paragraph text with optimal line length. The max-width is set to 65ch (characters), which is the sweet spot for readability. Studies show that 60-75 characters per line is easiest to read.

Strong text for emphasis. The body text uses fluid sizing: clamp(1rem, 1vw + 0.5rem, 1.125rem). This means it starts at 16px on mobile, scales with the viewport, and never exceeds 18px on large screens.

Small text also scales fluidly: clamp(0.875rem, 0.5vw + 0.5rem, 1rem)

Responsive Units in Action

Viewport Width (vw)

50vw = 50% of viewport width

Viewport Height (vh)

20vh = 20% of viewport height

REM Units

2rem padding (relative to root font-size: 16px = 32px)

CH Units (Character Width)

40ch max-width (40 characters wide - perfect for short text)

Clamp Function

Font size: clamp(1rem, 3vw, 2rem) - scales smoothly

Min/Max Functions

width: min(90%, 600px) - never wider than 90% OR 600px
